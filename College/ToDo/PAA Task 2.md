## **Problem**

Given multiples sorted array, remove the lowest value of the arrays.
By doing this multiple times, sum all the lowest value of each array, than print it.

## Example Input

```
3 2
3 4 5 6
2 2 3
4 1 2 3 4
```

removing by one time (2 - 1) the lowest values...

```
4 5 6
2 3
2 3 4
```
