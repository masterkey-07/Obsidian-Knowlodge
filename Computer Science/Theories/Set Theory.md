# Definitions

- ## Power of a Set

### $$A  = \{1,2,3\}$$
### $$2^A  = \{0,\{1\},\{2\},\{3\}, \{1,2\}, \{1,3\}, \text{etc...}\}$$

- ## Definition of a Set

### $$L = \{w | w \text{ is something...}\}$$
### $$L = \{w | w \ \epsilon \ A\}$$

- ## Product of Sets

### $$A \times B = \{(a,b)|a \ \epsilon \ A \text{ and }  b \ \epsilon \ B\}$$
### $$|A \times B| = |A| * |B|$$
