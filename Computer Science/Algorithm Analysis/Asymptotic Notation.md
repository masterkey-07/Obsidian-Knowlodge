# Symbols Meaning

 - ## **Theta**
#   $\theta(g(n)) = \{0 \le c_1 g(n) \le f(n) \le c_2 g(n)\}$

- ## **Big O**
#   $O(g(n)) = \{0 \le f(n) \le c g(n)\}$    

- ## **Little O**
#   $o(g(n)) = \{0 \le f(n) \lt c g(n)\}$     

- ## **Omega**
#   $\Omega(g(n)) = \{0 \le c g(n) \le f(n)\}$

- ## **omega**
#   $\omega(g(n)) = \{0 \le c g(n) \lt f(n)\}$
